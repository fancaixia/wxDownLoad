### wxDownLoad

小程序中文件下载  

此案例仅为案例演示 多个文件下载请自行调试

上面eee.doc 为要下载的文件

下面ee.doc 为下载完成后保存到本地的文件列表

### 项目启动

static-download为小程序项目，

node-download为node项目，node-download目录下upload目录下载的资源文件

若要启用node服务  cd/到node-download  然后cnpm install安装依赖  然后执行命令node app.js便可以访问node服务器资源
